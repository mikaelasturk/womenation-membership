Instructions and requirements for this week:

  - Landing Page: Create a landing page featuring 2 or 3 membership cards visible to all users.
  - Payments: Each card should include a "subscribe" button that directs users to a payment page.
  - Authentication: Include signup and login buttons in the top right corner. Use better-auth for authentication to ensure only authorized users can access their accounts.
